Hash function is a mathematical function mapping an arbitrarly long 
input $m$ into a fixed-size output $h$. The result of such operation
is called a _hash digest_.

$\operatorname{hash}(m) = h$

## Properties {#properties}

### Fixed values range {#fixed-values-range}

Every hash digest $h$ should have a defined, fixed length.

### Deterministic {#deterministic}

For a given input $m$ the same hashing function should always give the 
same value $h$.

### Uniform {#uniform}

Values of $h$ should be evenly distributed over the hash function's 
codomain.

## Examples: {#examples}

* Java's `Object::hashCode`
* Cassandra's partition key
* YouTube's URL shortener
