    HTTP Authentication

    Basic#

    caution

    The fact credentials are encoded does not provide any protection and therefore this kind of authencitation should not be used for communication over unencrypted protocol.

    Sandbox#

    Basic Authorization

    user, password → user:password → dXNlcjpwYXNzd29yZA==

    HTTP Request
    GET /index.html HTTP/1.1
    Host: www.example.com
    Authorization: Basic dXNlcjpwYXNzd29yZA==

    Bearer#

    HTTP Request
    GET /index.html HTTP/1.1
    Host: www.example.com
    Authorization: Bearer <token>

    An example of token-based authentication is JSON Web Token.
