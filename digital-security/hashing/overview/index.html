    Hashing Overview

    Hash function is a mathematical function mapping an arbitrarly long input mm into a fixed-size output hh. The result of such operation is called a hash digest.

    hash⁡(m)=h\operatorname{hash}(m) = h

    Properties#

    Fixed values range#

    Every hash digest hh should have a defined, fixed length.

    Deterministic#

    For a given input mm the same hashing function should always give the same value hh.

    Uniform#

    Values of hh should be evenly distributed over the hash function's codomain.

    Examples:#

    • Java's Object::hashCode
    • Cassandra's partition key
    • YouTube's URL shortener
